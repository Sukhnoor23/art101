Creating an web page:



1. First use an html to help create your website.
2. Then find domain name and host like github.
3. Pick your website address. Domain name example; yourname.com
4. Then start designing using stylesheets.
5. Add content through html and design a theme through lab.css.
6. Then view in the browser and inspect to make sure everything is looking how you want it.
7. Then you commit and push to the origin to publish it.
